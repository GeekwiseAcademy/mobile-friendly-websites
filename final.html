Final Project

Basics

  • Readability/intelligibility of Code
  • File/Directory structure
  • Proper Indentation/Nested Code
  • Everything is linked correctly (i.e. CDNs, local files, meta tags, link tags, etc.)
  • Did you incorporate Bootstrap correctly? (i.e. Is the Grid System properly implemented, what components did you use and why did you use those components?)
  • Is your site responsive?
  • At least one media query

Github

  • Must have a repository for your portfolio
  • Must have your portfolio hosted at [username].github.io

Design

  • Nav/Navbar
  • Jumbotron or Carousel
  • Accordion or thumbnails
  • At least three pages-worth of intelligable, coherent content that you created
  • Must include link to at least 1 site you've created
  • Use at least 4 social media/font awesome icons
  • Use at least 2 Google Fonts
  • Contact form that you styled
